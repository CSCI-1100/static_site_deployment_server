📖 How to Use the Static Site Deployment Server

Getting Started

Welcome! This tool helps you deploy your static websites so you can see them running live. Here's how to use it:

1. 📁 Organize Your Files

Before uploading, make sure your files are organized properly:

  • HTML files: index.html, about.html, contact.html, etc.
  • CSS files: All your stylesheets (.css files)
  • JavaScript files: All your scripts (.js files)
  • Images & Assets: Photos, logos, fonts, etc.

2. 📤 Upload Your Files

Use the dashboard to upload your files. You can either:

  • Drag and drop files into the appropriate sections
  • Click on a section to browse and select files

3. ✅ Check Your File References

Make sure your HTML files reference other files correctly:

<!-- ✅ Correct way to reference files -->
<link rel="stylesheet" href="css/style.css">
<script src="js/main.js"></script>
<img src="img/photo.jpg" alt="My photo">

<!-- ❌ Wrong - don't include your computer's path -->
<link rel="stylesheet" href="C:/Users/yourname/Desktop/style.css">

4. 🌐 View Your Website

Click the "View Website" button to see your live site! Your URL is:

https://csciauto1.etsu.edu:5000/{{ user.username }}

5. 💾 Export Your Files

Use the "Export ZIP" button to download all your files as a backup.

Common Issues

🚫 My images aren't showing

Check that:

  • Your image files are uploaded to the Images & Assets section
  • Your HTML uses src="img/filename.jpg"
  • The filename and extension match exactly (case-sensitive)

🎨 My CSS isn't working

Check that:

  • Your CSS files are uploaded to the CSS Files section
  • Your HTML uses href="css/style.css"
  • The CSS filename matches exactly

⚡ My JavaScript isn't working

Check that:

  • Your JS files are uploaded to the JavaScript Files section
  • Your HTML uses src="js/main.js"
  • The JavaScript filename matches exactly

Need More Help?

If you're still having trouble:

  • Double-check your file references in HTML
  • Make sure all files are uploaded to the correct sections
  • Ask your instructor for assistance
